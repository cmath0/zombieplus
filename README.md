## Zombieplus

Este é o repositório da aplicação usada como base para o projeto de automação com **Playwright** durante o curso "Playwright Zombie Edition". Esse projeto (Zombieplus) não é de minha autoria, apenas foi disponilizado no curso para o desenvolvimento do projeto de automação.

Você pode conferir o projeto de testes automatizados [aqui](https://github.com/cmath0/playwright-zombieplus-automation) :)
